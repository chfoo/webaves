Synopsis
========

**dump** [**-h**\ \|\ **--help**] [**-o**\ \|\ **--output**]
[**--overwrite**] <*input*>

Description
===========

Transform WARC files to JSON formatted output

Options
=======

**-h**, **--help**
   Print help information

**-o**, **--output** [default: -]
   Path to output file

**--overwrite** [default: false]
   Allow overwriting existing files

<*input*>
   Path to WARC file
