Synopsis
========

**dump** [**-h**\ \|\ **--help**] [**-o**\ \|\ **--output**] <*input*>

Description
===========

Transform WARC files to JSON formatted output

Options
=======

**-h**, **--help**
   Print help information

**-o**, **--output** [default: -]
   Path to output file

<*input*>
   Path to WARC file
